하나의 인터페이스
모든 교류

당사의 다중 교환 단말기는 모든 교환에 대한 간단한 인터페이스를 제공하며 자산 분석에서 증거금 거래까지 모든 도구를 제공합니다.

항상 온라인

매번 로그인하여 브라우저를 열 필요가 없습니다. 응용 프로그램은 항상 백그라운드에서 작동하고 신속하게 거래를 시작할 수 있습니다.

모든 계정

그는 다중 교환 터미널을 통해 사용자를 허용합니다. 반복적으로 전환하고 로그인 할 필요없이 여러 교환기를 동시에 쉽게 교환 할 수 있습니다.

다양한 기능 수익성 높은 거래를위한

단말기는 교환기로부터 직접 시장 데이터를 수신합니다. 가격 차트는 기술 분석을위한 전체 툴킷 및 데이터 세트와 함께 표시됩니다.

다양한 거래소의 총 유동성을 이용하면 전략을 가장 폭넓게 구현할 수있는 능력, 차익 거래 기회를 취하고, 변화하는 시장 기회. 이는 교환 거래의 웹 터미널보다 훨씬 빠른 API 거래의 이점입니다.

키없이 작업하기 API

OneExBit은 교환 계정 없이도 작업 할 수 있습니다.

분석 모드에서는 거래소의 모든 자산을 확인할 수 있습니다. 거래 간 빠른 전환으로 차트, 주문서, 최근 거래를 엽니 다.

여러 촛대 차트

하나, 두 개 또는 네 개의 차트를 열고 원하는대로 사용자 정의하십시오.

각 차트는 시장 분석에 도움이되는 실시간 데이터를 보여줍니다.

마진 거래

단말기는 BitMEX 거래에서 증거금 거래를 지원하고 있습니다.

한계 거래는 엄격한 규제의 적용을받으며 모든 거래소에서 지원하지 않습니다. 그러나 노련한 전문 레버리지 거래를 통해 매우 높은 수익을 올릴 수 있음을 인정해야합니다. 이러한 이유로 OneExBit 단말기는 확인 된 거래자에게 Poloniex 및 Bitmex와 같은이 서비스를 제공하는 모든 통합 거래소에서 마진 거래 도구를 제공합니다.
